"The Garden Party"

This is a movie poster for the short story, The Garden Party by Katherine Mansfield. I created vector art by tracing a found image on Adobe Illustrator. I also used the Shape Builder Tool in the program to smooth out shapes and create illustrations. After I finished the visuals of the poster, I experimented with fonts using the type tool.

Applications/Materials Used
